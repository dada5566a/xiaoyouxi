# 微信小游戏小程序流量主源码

## 📝 项目简介

本项目是一个开箱即用的微信小游戏小程序流量主源码，完全免费开源，帮助开发者快速搭建带流量主功能的小游戏。

## ✨ 项目特点

- **开箱即用** - 无需复杂配置，下载即可使用
- **完全免费** - 源码完全开源，无任何费用
- **简单易用** - 使用微信开发者工具直接打开即可运行
- **流量主集成** - 已集成微信流量主广告功能

## 🚀 快速开始

### 使用步骤

1. 下载本项目源码到本地
2. 打开微信开发者工具
3. 选择"导入项目"
4. 选择本项目文件夹
5. 开始开发或直接预览

### 环境要求

- 微信开发者工具（最新版本）
- 微信小程序账号（需开通流量主权限）

## 💡 功能特性

- 小游戏基础框架
- 流量主广告展示
- 激励视频广告
- Banner 广告位
- 插屏广告支持

## 📋 目录结构

```
├── pages/          # 页面文件
├── components/     # 组件文件
├── utils/          # 工具函数
├── images/         # 图片资源
├── app.js          # 小程序逻辑
├── app.json        # 小程序配置
└── app.wxss        # 小程序样式
```

## 🔧 配置说明

1. 在 `app.json` 中配置你的小程序信息
2. 在微信公众平台配置流量主广告位
3. 将广告位 ID 替换到对应页面代码中
